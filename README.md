## `tb_git_summary`

Collect information about a git repository and write it to file or `STDOUT`.

Information includes:
- Local directory
- Query date
- Branch
- Commit
- Origin's URL
- Diff

The output format looks like this:
~~~~
<key1>  : <value1>
<key2>  : <value2>
...     : ...
<keyN>  : <valueN>
----
<diff>
~~~~
